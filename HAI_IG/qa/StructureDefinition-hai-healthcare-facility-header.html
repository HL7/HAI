http://hl7.org/fhir/us/hai/StructureDefinition/hai-healthcare-facility

This profile records location information such as the facility id. The facility ID is an NHSN assigned faciltiy OID and if there is an extension it must be a value registered with NHSN.

This profile builds on Location.
