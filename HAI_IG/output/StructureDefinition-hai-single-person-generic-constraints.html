D.4.1 StructureDefinition: hai-single-person-generic-constraints

D.4.1.1 Formal Views of Profile Content

The official URL for this profile is: 

http://hl7.org/fhir/us/hai/StructureDefinition/hai-single-person-generic-constraints

This profile records the constraints for HAI single-person reports. It is used by all numerator reports covered by this guide. It is also used by the Procedure Denominator Report.

A single-person report must conform to the Healthcare Associated Infection Report profile. In addition, key data about the patient are recorded in the HAI Patient profile referenced in the subject.

The report-specific requirements specify the encounter data to record for each single-person report type. These data always include an identifier for the reporting facility, and usually include the encounter type (inpatient or outpatient), the admission date or encounter date, and the facility unit identifier and type.

This profile builds on Healthcare Associated Infection Report.

This profile was published on Sat Sep 23 15:59:00 AEST 2017 as a draft by Lantana Consulting Group.
