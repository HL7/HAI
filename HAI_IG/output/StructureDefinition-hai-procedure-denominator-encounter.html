D.4.1 StructureDefinition: hai-procedure-denominator-encounter

D.4.1.1 Formal Views of Profile Content

The official URL for this profile is: 

http://hl7.org/fhir/us/hai/StructureDefinition/hai-procedure-denominator-encounter

Key encounter data: The encounter type (inpatient or outpatient) is required. If the patient was an inpatient, a value for admission date is required.

This profile builds on Encounter.
