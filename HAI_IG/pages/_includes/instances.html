This guide defines the following instances of the Questionnaire and QuestionnaireResponse profiles. The Questionnaire instance defines the HAI form and the Questionnnaire instance is an example of a completed QuestionnaireResponse to the Questionnaire.

Instance Name Description
Questionnaire/hai-questionnaire-los-event This instance represents the LOS Event form as a FHIR Questionnaire.
Questionnaire/hai-questionnaireResponse-los-event This profile represents an example response to the LOS Event form Questionnaire as a FHIR QuestionnaireResponse.
